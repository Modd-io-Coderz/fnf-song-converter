Having problems? Some of these may help:
Custom Song Errors are only supported on v11 and later. When modding, you may want to update to this version, or later.

[v11+] Custom song error: No beatmap loaded

Your beatmap is either completely empty or is not loaded in the songHandler. You probably didn't do step 3 or 4 properly.

[v11+] Custom song error: Beatmap is invalid

Your beatmap isn't a valid beatmap. Did you import the right file into Scratch? Delete your current list, follow step 3 onwards, and, if that didn't help, read the whole guide again.

[v11+] Custom song error: 2P not set

Redo step 4, and follow everything closely. You didn't set a proper 2P character.

[All versions] The stage is blank / one or both of the characters aren't showing up

You set an invalid stage/character, or didn't set one at all. Go back to step 4.

[All versions] No audio is being played

You either didn't set up your audio correctly, or didn't load any audio files at all. Check the second half of step 4.

[< v11] My song isn't loading properly / random notes are being played

This is probably because you didn't load your beatmap. See step 4.

[< v11] I set a stage, but the background is still blank

This is a problem plaguing pre-v11 versions of my project. Either:
  • Update to v11
  • Go to the selected stage's sprite and add an exception for your song (search for "if currSongName == ___")
